Instructions:
i) Compilez le code avec gcc en utilisant: 
gcc -pthread SleepingTeachingAssistant.c -o SleepingTeachingAssistant

ii) Exécutez le programme en spécifiant le nombre d'étudiants comme argument. Par exemple, pour simuler 5 étudiants :
./SleepingTeachingAssistant n

ici n sera le nombre d' 'etudiants.

par e.g pour simuler 5 etudiants
./SleepingTeachingAssistant 5 

------------------------------
Assumptions:
Le nombre de chaises dans le couloir est fixé à 3. Cela n'était pas spécifié dans les instructions, nous avons donc fait cette supposition pour simplifier l'implémentation.

-------------------------------
Note:
Le programme est conçu pour simuler un environnement infini où les étudiants alternent continuellement entre la programmation et la demande d'aide. Pour arrêter l'exécution du programme, utilisez Ctrl+C dans le terminal.
